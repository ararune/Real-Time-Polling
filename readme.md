
## General info

> This is a real time polling app using Node.js, Express, Pusher, MongoDB and CanvasJS

## Quick start


```powershell
# Install dependencies
npm install

# Run dev server localhost:3000
npm run dev

# Start
npm start
# localhost:3000/poll to view the DB json
```
